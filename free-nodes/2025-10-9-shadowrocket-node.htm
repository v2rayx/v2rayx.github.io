一、什么是 Quantumult X？

Quantumult X 是一款为 iOS 平台设计的网络优化工具，广泛应用于需要科学上网、提高网络速度与隐私保护的用户。通过 Quantumult X，用户可以实现跨地区的访问、加速网络、管理流量等多种功能。

1.1 Quantumult X 的功能特点

  • 易于使用的用户界面：Quantumult X 提供简洁直观的界面，即使是没有技术背景的用户也能够快速上手。

  • 高度自定义配置：用户可以根据需求自定义各种规则，从而为自己量身定制最佳的网络体验。

  • 实时流量监控：内置流量监控功能，可以实时查看网络数据的使用情况，帮助用户控制上网行为，避免浪费流量。

二、Quantumult X 的下载安装

在开始配置 Quantumult X 之前，首先需要将其下载并安装到你的 iOS 设备上。安装过程非常简单，下面是详细步骤：

  1. 打开 Apple App Store。

三、Quantumult X 的基础设置

3.1 设置配置文件

首次打开 Quantumult X 时，你将被引导设置配置文件。根据你的使用需求，选择合适的配置模板至关重要。常见的配置选项有：

  • 全球节点：这是一个广泛适用的配置，能够显著提升网络的速度和稳定性。它通常适用于经常需要跨地区访问网站或应用的用户。

选择完合适的配置模板后，你将完成基础的设置，接下来可以添加代理以进一步优化网络体验。

3.2 添加代理

Quantumult X 提供强大的代理功能，允许你通过输入代理信息来提升网络连接的稳定性和安全性。以下是添加代理的步骤：

配置完成后，代理将立即生效，你的网络流量将通过指定的服务器进行传输。

  • 白名单设置：将重要的或常用的网址添加到白名单，以避免这些网址被误拦截，确保访问的顺畅性。

设置脚本的步骤如下：

  3. 在使用脚本前，务必确保脚本无误并经过充分测试，以免影响网络连接或出现其他问题。

FAQ

Nekobox 是否支持跨客户端节点管理？
支持。Nekobox 可以独立管理 Shadowsocks、Clash 等客户端节点，互不干扰。结合策略组，可实现多客户端流量分流和网络优化，提高节点管理效率和使用便利性。
如何在 Nginx 上为 Vmess/WS 配置反向代理与 TLS？
在 Nginx 中配置 HTTPS 站点并将特定路径（如 /ray）反向代理到 V2Ray 的 WebSocket 端口，确保 proxy_set_header 设置正确（Host、X-Real-IP 等）。同时使用 certbot 获取并自动续期 TLS 证书以维护安全。
Netch 是否可以代理局域网设备？
可以。Netch 支持启用局域网共享功能，使同一网络下的其他设备通过主机的代理访问外网。需确保防火墙允许入站连接，且代理端口在局域网中开放。
Matsuri 可以用于 Android 吗？
Matsuri 主要针对桌面平台开发，目前并未推出 Android 版本。但可使用其配置文件在兼容的客户端（如 Nekobox 或 Clash Android）中加载以获得类似体验。
V2Ray 服务器启动后提示端口被占用，该如何处理？
先用 netstat 或 ss 工具查看占用该端口的进程（例如 `ss -ltnp`），确认是旧进程或其他服务占用。停止占用进程或修改 V2Ray 配置端口，然后重启服务并检查防火墙规则是否开放了新端口。
Mihomo 节点分组功能有什么优势？
分组管理可按延迟、地区或用途分类节点，并结合策略组实现自动分流。用户可快速选择最优节点，提高访问速度和稳定性，方便管理大量节点并优化维护效率。
如何在 V2Ray 中自定义路由规则？
在 V2Ray 中可以通过修改 config.json 文件中的 routing 部分来自定义路由规则。用户可以根据 IP 段、域名或地理位置来决定流量的去向，从而实现自动分流或访问加速的效果。
Sing-box 如何实现精确端口分流？
Sing-box 支持按端口匹配流量，将不同类型应用流量分配到指定节点。结合策略组，可提高访问速度和网络稳定性，实现精准分流和优化用户体验。
VLESS 的 Flow 参数有什么作用？
Flow 参数用于控制 VLESS 的传输模式，如 XTLS Vision 或 Reality 模式。它能影响性能、安全性与兼容性，需根据客户端支持情况正确匹配配置。
Clash 如何通过规则文件优化节点分流？
Clash 支持自定义规则文件，可按域名、IP 或应用类型分流流量。结合策略组使用，可将不同类型的流量自动分配到最优节点，提高访问速度和网络稳定性，减少延迟和卡顿。
